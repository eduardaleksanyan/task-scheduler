# Setup

1. Run docker-compose up -d to start the necessary services

# Backend
1. Navigate to the backend directory
2. npm install
3. Rename example.env to .env
5. Update the .env file with suitable values. You can keep the existing values and only change the port if it is already in use
6. Start the backend service: 
### `npm start`

# Frontend
1. Navigate to the backend directory
3. npm install
4. Rename example.env to .env
5. Update the .env file with suitable values. You can keep the existing values and only change the port if it is already in use
6. Start the frontend service:
### `npm start`
